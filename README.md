# GNU C Language Manual

This repository is a fork of [Vernon Grant's Markdown conversion](https://github.com/VernonGrant/gnu-c-language-manual) of **Richard Stallman's GNU C Language Intro and Reference**, featuring an ongoing translation into Azerbaijani.

The text is periodically synchronized with the upstream changes from the official GNU [repository](https://git.savannah.nongnu.org/cgit/c-intro-and-ref.git/). 

If you find any flaws, typos, or passages in the translation that are unclear, please consider opening an issue here so they can be addressed.
